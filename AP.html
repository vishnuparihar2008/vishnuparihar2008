Unit I — Quantum Mechanics

de Broglie & Wave-Particle Duality

de Broglie wavelength λ = h / p = h / mv
in terms of KE λ = h / √(2mE)
accelerated by V volts λ = h / √(2meV)
electron shortcut λ = 1.226 / √V Å (V in volts)
wave–particle relation E = hν = hc / λ = ħω
wave relations c = λν , ω = 2πν , k = 2π/λ
momentum–wavevector p = ħk
photon energy E = hν = pc → p = h/λ
relativistic KE E² = (pc)² + (m₀c²)²

Uncertainty Principle & Wave Function

position–momentum Δx · Δp ≥ ħ/2 (= h/4π)
energy–time ΔE · Δt ≥ ħ/2
normalization ∫ |ψ|² dx = 1 (over all space)
probability density P(x) = |ψ|² = ψ* ψ
expectation of x ⟨x⟩ = ∫ ψ* x ψ dx
expectation of p ⟨p⟩ = ∫ ψ* (−iħ d/dx) ψ dx
momentum operator p̂ = −iħ ∂/∂x
energy operator Ê = iħ ∂/∂t
conditions on ψ single-valued, finite,
continuous, → 0 at ±∞

Schrödinger Equation

Time-dependent (general) iħ ∂ψ/∂t = −(ħ²/2m)∇²ψ + Vψ
Time-independent (steady state) −(ħ²/2m) d²ψ/dx² + Vψ = Eψ
Standard form (free region, V=0) d²ψ/dx² + k²ψ = 0 where k² = 2mE/ħ²
General solution (V=0) ψ = A sin(kx) + B cos(kx)
Forbidden region (E < V) d²ψ/dx² − α²ψ = 0 where α² = 2m(V−E)/ħ² ψ = C e^(αx) + D e^(−αx)
Eigen value eq. Ĥ ψ = E ψ

Particle in 1D Infinite Well (Box)

Setup: V=0 for 0 ≤ x ≤ L, V=∞ outside
eigen energy Eₙ = n²h² / 8mL² = n²π²ħ²/2mL²
n = 1,2,3... E₁ = h²/8mL² (ground state)
eigen function ψₙ = √(2/L) · sin(nπx/L)
energy shortcut Eₙ = n² E₁
energy difference E_{n+1} − Eₙ = (2n+1) · E₁
expectation ⟨x⟩ ⟨x⟩ = L/2 (all states)
expectation ⟨p⟩ ⟨p⟩ = 0 (all states)
quantization cond. kL = nπ → k = nπ/L
QHO zero-point E E₀ = ħω/2 (n=0 state)

Phase & Group Velocity

phase velocity vₚ = ω/k = E/p
group velocity vg = dω/dk = particle velocity v
non-relativistic vₚ = v/2 , vg = v
relativistic relation vₚ · vg = c²
dispersion relation ω = ħk² / 2m (non-rel.)
vg derivation E = p²/2m → ħω = ħ²k²/2m dω/dk = ħk/m = p/m = v ✓

Quantum Tunneling

transmission prob. T ≈ e^(−2k₂L)
decay constant k₂ k₂ = √(2m(V₀−E)) / ħ
alternate form 2k₂L = (4πL/h)√(2m(V₀−E))
condition for tunneling E < V₀ (particle energy < barrier)
region I wave (x<0) ψ₁ = A e^(ik₁x) + B e^(−ik₁x)
region II (barrier) ψ₂ = C e^(k₂x) + D e^(−k₂x)
region III (x>L) ψ₃ = F e^(ik₁x)
k₁ in free region k₁ = √(2mE) / ħ
Unit II — Statistical Mechanics

MB / BE / FD Distribution Functions

Maxwell-Boltzmann f_MB(E) = A · e^(−E / k_BT) — classical, distinguishable particles
Bose-Einstein f_BE(E) = 1 / [e^((E−μ)/k_BT) − 1] — bosons (integer spin)
Fermi-Dirac f_FD(E) = 1 / [e^((E−E_F)/k_BT) + 1] — fermions (half-integer spin)
FD at T = 0 f = 1 if E < E_F , f = 0 if E > E_F
FD at E = E_F f(E_F) = 1/2 (at any temperature T > 0)
MB limit BE and FD → MB when e^((E−μ)/k_BT) >> 1 (high T, low density)
bosons Photon (spin 1), phonon, α-particle (He-4), π meson, He-4 atom
fermions Electron, proton, neutron, He-3, positron (all spin = ½)

Average Energies

MB average energy ⟨E⟩ = (3/2) k_BT

Fermi Energy

E_F = (ħ²/2m)(3π²n)^(2/3) = (h²/2m)(3n/8π)^(2/3)
n = free electrons/m³ n = (ρ × N_A × Z) / M
Fermi temperature T_F = E_F / k_B
Fermi momentum p_F = ħ(3π²n)^(1/3)
Fermi velocity v_F = p_F / m_e = ħ(3π²n)^(1/3)/m
free electrons in K Given E_F: n = (8π/3)(2mE_F/h²)^(3/2)
mean energy T=0 ⟨E⟩₀ = (3/5) E_F

Planck's Radiation Law & Black Body

Planck (per freq.) u(ν,T) = (8πhν³/c³) × 1/[e^(hν/k_BT) − 1]
Planck (per wavelength) u(λ,T) = (8πhc/λ⁵) × 1/[e^(hc/λk_BT) − 1]
Wien's displacement λ_max · T = 2.898 × 10⁻³ m·K
Stefan-Boltzmann E = σT⁴ , σ = 5.67 × 10⁻⁸ W m⁻² K⁻⁴
Wien's law (limit) hν >> k_BT → u(ν,T) ≈ (8πhν³/c³)e^(−hν/k_BT)
Rayleigh-Jeans (limit) hν << k_BT → u(ν,T) ≈ (8πν²/c³)k_BT
Unit III — Crystal Structure

Crystal Structure

Bragg's law 2d sinθ = nλ
interplanar spacing d_hkl = a / √(h²+k²+l²) (cubic)
common planes d₁₀₀=a , d₁₁₀=a/√2 , d₁₁₁=a/√3
HCP c/a ratio c/a = √(8/3) ≈ 1.633
atoms per cell SC=1 , BCC=2 , FCC=4 , HCP=6
BCC: touch condition 4r = a√3 → r = a√3/4
FCC: touch condition 4r = a√2 → r = a√2/4
packing fraction SC = 52% BCC = 68% FCC = 74%
lattice constant a = (nM / ρN_A)^(1/3) [n=atoms/cell]

Crystal Defects

Schottky defect conc. nₛ = N · e^(−Eₛ/2k_BT)
Frenkel defect conc. n_F = √(N·Nᵢ) · e^(−E_F/2k_BT)
Frenkel proportionality n_F ∝ √(N · Nᵢ)
Schottky: vacancy pair Atom leaves lattice site → surface. Charge-neutral pairs in ionic crystals.
Frenkel: interstitial Atom → interstitial site. N = lattice sites, Nᵢ = interstitial sites.

Miller Indices — Quick Rules

Unit IV — Semiconductors & Band Theory

Semiconductors

intrinsic carrier conc. nᵢ = √(Nc·Nv) · e^(−Eg/2k_BT)
law of mass action n · p = nᵢ² (at any temperature)
Fermi level (intrinsic) E_F = (Ec + Ev)/2 + (3k_BT/4)·ln(mₕ*/mₑ*)
if mₑ* = mₕ* E_F = (Ec + Ev)/2 = midgap exactly
electron density n = Nc · e^(−(Ec−E_F)/k_BT)
hole density p = Nv · e^(−(E_F−Ev)/k_BT)
effective DOS Nc = 2(2πmₑ*k_BT/h²)^(3/2) , Nv = 2(2πmₕ*k_BT/h²)^(3/2)
